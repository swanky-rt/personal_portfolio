# Aarti Kumari – Portfolio

React + Vite portfolio for Aarti Kumari (Ex-PayPal Software Engineer 2, MS CS – Data Science @ UMass Amherst).

## Getting Started

```bash
npm install
npm run dev
```

## Images

- Place Aarti's main profile photo as: `public/aarti.jpg`
- Place gallery photos under: `public/gallery/` (e.g. `public/gallery/hackathon_uai.jpg`)

The app expects `/aarti.jpg` and `/gallery/<filename>`.

## Build

```bash
npm run build
npm run preview
```
